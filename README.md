# Script-Edit-Plugin for "Evil Islands" (TM) game script (can work with Zone View, ...) (c) Atm, 2023
Внешний редактор скрипта для игры "Проклятые Земли".
# Установка:
1. Zone View:
  пропишите в ZoneView.ini
  ScriptEditor="путь до файла\Script Edit Plugin.exe"

# Использование:
Перетащите текстовый файл со скриптом или запустите в автоматическом режиме через Zone View.

Примечание: Зонвью ждёт, пока скриптэдитор не закроется, после чего считывает файл скрипта.
Если вы закрыли скриптэдитор без сохранения - изменения не применятся! Так-же плохая идея открывать другой .mob с открытым скриптэдитором - по вполне очевидным причинам.

Все хоткеи вынесены в основное меню приложения, можете ознакомиться с ними сразу.

# Условия использования
1. Программа нахдится в стадии прототипа, распространяется "как есть", авторы(ы) не несут гарантий работоспособности и поддержки.
2. Вы можете свободно её использовать со следующими ограничениями:
    2.1. Запрещено использование в коммерческих продуктах и/или с прямой целью получения прибыли (например, платные модификации, инструменты с подпиской).
    2.2. Запрещено распространение без указания оригинального авторства и данных ограничений.
3. Автор(ы) имеют право:
    а. Запрещать использование программы определённому кругу лиц/организаций, без публичного объяснения причин. При этом результат работы программы, сделанный до даты запрета, остаётся за пользователями.
    б. Полностью изымать программу из открытого доступа.
